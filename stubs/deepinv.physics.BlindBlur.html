BlindBlur

class deepinv.physics.BlindBlur(kernel_size=3, padding='circular', **kwargs)[source]

Bases: Physics

Blind blur operator.

If performs

\[y = w*x\]

where \(*\) denotes convolution and \(w\) is an unknown filter. This class uses torch.conv2d for performing the convolutions.

The signal is described by a tuple (x,w) where the first element is the clean image, and the second element is the blurring kernel. The measurements y are a tensor representing the convolution of x and w.

Parameters:

  • kernel_size (int) – maximum support size of the (unknown) blurring kernels.

  • padding (str) – options are 'valid', 'circular', 'replicate' and 'reflect'. If padding='valid' the blurred output is smaller than the image (no padding) otherwise the blurred output has the same size as the image.
