class deepinv.physics.Pansharpen(img_size, factor=4, noise_color=GaussianNoise(), noise_gray=GaussianNoise(), filter='gaussian', device='cpu', padding='circular', **kwargs)[source]ÔÉÅ

Bases: LinearPhysics

Pansharpening forward operator.

The measurements consist of a high resolution grayscale image and a low resolution RGB image, and are represented using deepinv.utils.TensorList, where the first element is the RGB image and the second element is the grayscale image.

By default, the downsampling is done with a gaussian filter with standard deviation equal to the downsampling, however, the user can provide a custom downsampling filter.

It is possible to assign a different noise model to the RGB and grayscale images.

Example usage:

import deepinv
import torch

x = torch.randn(1, 3, 256, 256)
physics = deepinv.physics.Pansharpen(img_size=x.shape[1:], device=x.device)

y = physics(x) # returns a TensorList with the RGB and grayscale images

x_adj = physics.A_adjoint(y)
x_pinv = physics.A_dagger(y)

deepinv.utils.plot([y[0], y[1], x_adj, x_pinv, x], titles=['low res color', 'high res gray',
                                                          'A_adjoint', 'A_dagger', 'x'])

  • factor (int) ‚Äì downsampling factor.

  • noise_color (torch.nn.Module) ‚Äì noise model for the RGB image.

  • noise_gray (torch.nn.Module) ‚Äì noise model for the grayscale image.

  • filter (torch.Tensor, str, NoneType) ‚Äì Downsampling filter. It can be ‚Äògaussian‚Äô, ‚Äòbilinear‚Äô or ‚Äòbicubic‚Äô or a custom torch.Tensor filter. If None, no filtering is applied.

  • padding (str) ‚Äì options are 'valid', 'circular', 'replicate' and 'reflect'. If padding='valid' the blurred output is smaller than the image (no padding) otherwise the blurred output has the same size as the image.
